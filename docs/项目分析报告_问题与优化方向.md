## 二、当前项目整体架构总览

### 后端（FastAPI + SQLite）

```
backend/
├── main.py                 # FastAPI 入口，CORS，6 个 API Router
├── config.py               # 环境变量（LLM/Embedding/路径/端口）
├── core/
│   ├── agent.py            # BaseAgent（load_prompt/stream_llm/call_llm_json）
│   ├── context.py          # UnifiedContext 数据类
│   ├── orchestrator.py     # 总控调度（10 capability → Agent）
│   └── stream_bus.py       # SSE 事件流（asyncio.Queue）
├── agents/                 # 10 个 Agent
│   ├── chat_agent.py       # RAG 对话 + 文件/图片处理
│   ├── deep_solve.py       # 三阶段深度求解（Plan→Solve→Write）
│   ├── profiler.py         # 画像提取
│   ├── profile_builder.py  # 8 步引导式画像构建
│   ├── diagnostic.py       # 学习诊断
│   ├── resource_planner.py # 资源规划
│   ├── generator.py        # 内容生成 + 安全审查 + 持久化
│   ├── path_planner.py     # 路径规划 + 自适应调整
│   ├── evaluator.py        # 评估 → 掌握度更新
│   └── safety.py           # 安全审查
├── services/
│   ├── database.py         # SQLite 5 表
│   ├── profile_service.py  # 画像管理
│   ├── mastery_service.py  # 掌握度追踪（衰减机制）
│   ├── resource_service.py # 资源存储（extra 字段）
│   ├── session_service.py  # 会话管理 + 自动摘要 + 画像刷新
│   ├── rag_service.py      # RAG 双模检索（多课程知识库）
│   ├── knowledge_graph.py  # NetworkX DAG（19 节点）
│   ├── llm_service.py      # LLM（重试 + Token 统计 + 响应清洗）
│   └── prompt_manager.py   # Prompt 模板
├── api/                    # 5 个 Router
│   ├── chat.py             # /api/chat（SSE）+ /sync + /sessions + /history + /image
│   ├── profile.py          # /api/profile
│   ├── resources.py        # /generate + /list + /detail + /delete + /upload-file + /knowledge/upload
│   ├── path.py             # /plan + /adjust + /graph
│   └── evaluation.py       # /submit + /parse-quiz + /diagnose + /mastery
├── prompts/                # Prompt 模板目录
│   ├── chat/               # 对话 Agent 模板
│   ├── solve/              # 深度求解 Agent 模板（plan/solve/write）
│   ├── generator/          # 生成 Agent 模板
│   ├── diagnostic/         # 诊断 Agent 模板
│   └── ...                 # 其他 Agent 模板
└── data/
    ├── knowledge_bases/
    │   ├── python_programming.json
    │   └── knowledge_graph.json
    └── uploads/            # 图片上传目录
```

### 前端（原生 JS + marked.js + highlight.js + mermaid.js）
