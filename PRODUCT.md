# 智学助手 - 产品定义

## 产品愿景

智学助手是面向高等教育场景的个性化学习智能体系统。通过构建学生画像，为每位学习者提供定制化的资源推荐、路径规划和智能辅导，让学习从"千人一面"转向"因材施教"。

## 目标用户

高校在读学生，主修计算机科学、人工智能或相关专业，需要系统化学习 AI/ML 课程并希望获得个性化学习指导。

## 核心功能

### 首页仪表盘
基于学习画像的个性化推荐，展示今日推荐资源、课程进度概览。时辰问候语增强亲和力。

### 学习画像
可视化展示学生的知识掌握度、学习活跃度、认知风格和薄弱知识点。使用圆环图和进度条直观呈现数据。

### 资源中心
聚合多模态学习资源（文档、视频、题库、代码），支持按类型筛选，不同资源类型有差异化展示。

### 学习路径
基于画像生成的阶段性学习计划，时间线可视化展示已完成、进行中和待开始的阶段。

### 智能辅导
对话式学习助手，支持关键词匹配的智能回复，提供预设建议提示降低使用门槛。

## 技术架构

- 纯前端实现，无需后端服务
- 模块化架构：hash 路由 + 模块生命周期管理
- CSS 自定义属性驱动的设计系统
- OKLCH 色彩空间
